HTML

HTML (HyperText Markup Language) is the most basic building block of the Web.

It defines the meaning and structure of web content.

Other technologies besides HTML are generally used to describe a web pages.

CSS

Cascading Style Sheets is a style sheet language

used for specifying the presentation and styling of a document written in a markup language such as HTML or XML.

CSS is a cornerstone technology of the World Wide Web,

alongside HTML and JavaScript.

JavaScript

JavaScript, often abbreviated as JS , is a programming language and core technology of the Web,

alongside HTML and CSS.

99% of websites use JavaScript on the client side for webpage behavior.

Web browsers have a dedicated JavaScript engine that executes the client code.
